About HTML

What is HTML?

HTML stands for Hyper Text Markup Language. It is a standard markup language to develop a webpage.It describes the structure of a Web page which can be viewed by anyone connected to the internet.

Feature of HTML:

  • It is very easy and simple language.
  • HTML is widely used language on the web.
  • It create a state website only.
  • It has a lot of formatting tags.
  • HTML is a case-insensitive language.

      • Example:

      
    <!DOCTYPE html>
      <html>
        <head>
          <title>Order list</title>
        </head>
        <body>
          <h1>input devices </h1>
          <ol>
          <li>Monitor </li>
          <li>Keyboard </li>
        </body>
      </html>

      Example Demonstrates:

      This example demonstrates the basic structure of an HTML document and how to use headings and lists to organize content. Browsers use these elements to display the web page correctly, showing the heading "input devices" followed by a numbered list of "Monitor" and "Keyboard."

      Element Purpose
      <!DOCTYPE html> Declares the document type and HTML version
      <html> Root element of the HTML page
      <head> Contains meta-information and title
      <title> Sets the page title in the browser tab
      <body> Contains all visible content
      <h1> Main heading
      <ol> Ordered (numbered) list
      <li> List item inside ordered list
