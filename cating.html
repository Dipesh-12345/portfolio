About CSS

What is CSS?

Cascading Style Sheets is a language used to control the visual presentation of web pages. While HTML structures the content (e.g., headings, paragraphs, forms), CSS determines how that content looks and feels. It's what makes websites visually appealing and user-friendly!

Type of CSS:

  • Inline CSS
  • Internal CSS
  • External CSS

Features of CSS:

  • Separation of Content and Presentation.
  • Cascading and Specificity.
  • . Reusability and Efficiency

CSS Selector:

CSS selectors are used to fonding or selecting the HTML element that you want to style. The CSS uses different types of elements.

Element Selector:

The element selector HTML elements based on the element name.

Example:


/* Element Selector */
p{
  color: red;
  text-align: center;
}

Id Selector:

Id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page. The sign(#) is used to define the id selector.

Example:

Here a simple example how to use id selector:


/* id Selector */
#para{
  margin:20px;
  font-size: 20px;
  color: blue;
}

Class Selector:

The class selector selects HTML elements with a specific class attribute.Sign (.) is used to define the class selector.

Example:

Here a example how to use class selector:


/* Class Selector */
.text {
    text-align: center;
    color: green;
    border-radius: 8px;
    box-shadow: 0 4px 6px 5px rgba(0,0,0,0.2);
    padding: 2rem;
    margin:20px ;
    margin-top: 5px;
}
