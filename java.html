About JavaScript

What is JavaScript?

JavaScript is a dynamic sscripting language (website development environment). It lightweight and most commonly used as a part of web pages.
JavaScript code can be inserted into any HTML page, and it can be executed by all types of web browsers.
JavaScript is embedded between <script>.... </script> HTML tags. But you can place anywhere in your web page, It is normally recommended that you should keep it within the Head tags.

Features of JavaScript:

  • It is a light-weighted and interpreted language.
  • It is structured programming language. It follows the syntax and structure of the C programming language.
  • It is case-sensitive language.
  • It support in various operating system including Windows, Mac OS etc.

JavaScript Data types:

Primitive Data Types

It can hold one value at a time. There are five types of primitive data types in JS. They are follows:

Data Type Description
String represents a sequence of characters example, "hello"
Number represents numeric value example, 100
Boolean represents Boolean value either false or true
Undefined represent an undefined value
Null represents null i.e, no value at all

Non-primitive Data Types

It can hold collections of values and more complex entities. Non-primitive data types are as follows:

Data Type Description
Object represents an instance through which we can access members
Array represents a group of similar values
RegExp represents a regular expression
