About Bootstrap

What is Bootstrap?

Bootstrap is a free and open-source CSS framework used for creating responsive, mobile-first websites. It includes pre-designed components like buttons, forms, navigation bars, and more, making web development faster and easier.

Example:

Here a simple example adding bootstrap:


      <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Bootstrap Example</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>

<div class="container text-center mt-5">
  <h1 class="text-primary">Hello, Bootstrap!</h1>
  <button class="btn btn-success mt-3">Click Me</button>
</div>

</body>
</html>

Example Demonstrates:

This example shows how easy it is to style headings, buttons, and layout using Bootstrap classes without writing custom CSS.

  • <link> - This tag loads Bootstrap CSS file from a CDN, which styles the page automatically.
  • .container - Adds horizontal padding and centers the page content.
  • .text-center - Centers the text inside the container.
  • .text-primary - Colors the heading text blue, using Bootstrap's "primary" color.
  • .btn and .btn-success - Styles the button with Bootstrap's green ("success") style.
  • mt-5 and mt-3 - Add top margins (spacing) to the heading and button.
